## ⬇️ Скачать

👉 **[Последняя версия → Releases](https://github.com/KUBA-KU/vela-browser/releases/latest)**

> Скачай `VELA Browser Setup x.x.x.exe` и запусти установщик.
> Обновления устанавливаются **автоматически** при запуске браузера.

---

## ✨ Возможности

- 🚀 Быстрый запуск и работа
- 🛡️ Встроенная блокировка рекламы и трекеров
- 🎨 Тёмная/светлая тема
- 🧩 Поддержка расширений и пользовательских скриптов
- 🔄 Автоматические обновления через GitHub
- 📌 Жесты мышью, боковая панель, история, закладки

---

## 🔄 Автообновление

VELA Browser проверяет обновления при каждом запуске.
Когда выходит новая версия — браузер скачивает её в фоне и предлагает перезапуститься.
